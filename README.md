# Bazi Streamlit App

八字命理分析 Streamlit 应用，包含完整的命盘分析、流年预测、健康解读、职涯规划等功能。

## 功能特性

- 📋 **基礎分析** - 完整的八字命盘分析
- 📊 **數據版圖** - 可视化数据展示
- 📅 **流年預演** - 流年运势预测
- 👨‍👩‍👧‍👦 **家族分析** - 家族关系分析
- 🎮 **戰略分析** - 战略决策分析
- 🏥 **健康解讀** - 健康能量分析
- 🎓 **進學之道** - 学业规划分析
- 🗺️ **職涯規劃** - 职业发展战略规划（新增）
- 🌟 **天象解讀** - 天象分析
- 💕 **情感解碼** - 情感关系分析
- 💰 **財富戰略** - 财富战略规划
- ⚔️ **RPG 裝備** - 角色扮演风格分析

## 安装

```bash
pip install -r bazi-master/requirements.txt
```

## 运行

```bash
cd bazi-master
streamlit run streamlit_app.py
```

## 人格分析功能

本項目包含完整的「12月令 × 12時辰」人格氣象分析總表，共144種組合。每個組合包含：

- **氣候調候**：五行平衡與調候分析
- **五行十神互動**：十神關係與能量流動
- **人格結構**：內在性格特質
- **社會表象**：外在表現與人際關係
- **潛在課題**：成長方向與注意事項

相關文件：
- `streamlit_app.py` - 主應用（包含完整 personality_matrix）
- `personality_matrix_fixed.py` - 格式修復示例
- `check_format.py` - 格式檢查工具
- `格式問題修復指南.md` - 詳細修復指南
