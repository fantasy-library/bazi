# Bazi Streamlit App

八字命理分析 Streamlit 应用，包含完整的命盘分析、流年预测、健康解读、职涯规划等功能。

## 功能特性

- 📋 **基礎分析** - 完整的八字命盘分析
- 📊 **數據版圖** - 可视化数据展示
- 📅 **流年預演** - 流年运势预测
- 👨‍👩‍👧‍👦 **家族分析** - 家族关系分析
- 🎮 **戰略分析** - 战略决策分析
- 🏥 **健康解讀** - 健康能量分析
- 🎓 **進學之道** - 学业规划分析
- 🗺️ **職涯規劃** - 职业发展战略规划（新增）
- 🌟 **天象解讀** - 天象分析
- 💕 **情感解碼** - 情感关系分析
- 💰 **財富戰略** - 财富战略规划
- ⚔️ **RPG 裝備** - 角色扮演风格分析

## 安装

```bash
pip install -r bazi-master/requirements.txt
```

## 运行

```bash
cd bazi-master
streamlit run streamlit_app.py
```
